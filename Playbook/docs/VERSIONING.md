# Governance Versioning

Current Governance Version: **v0.3.5**

## Version bump rules
- **MAJOR**: Breaking governance change (removes or redefines enforced contracts).
- **MINOR**: Adds a new guardrail or pattern with new enforceable scope.
- **PATCH**: Clarity, organization, or non-semantic edits.
