# Fitness Truth Pack

This folder is the deterministic, in-repo context pack for Fitness as a governed ecosystem participant.

## Contents

- `identity.json`: canonical app identity metadata.
- `governance-loop.md`: required governed loop and seam constraints.
- `signals.json`: canonical signal list and deterministic routing metadata.
- `state-snapshots.json`: inspectable app truth surface.
- `actions-and-receipts.json`: bounded action contract and declared receipt mappings.

All artifacts are human-readable and intended to be inspected directly in review workflows.
